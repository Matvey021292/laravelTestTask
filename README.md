Clone repository
-------------------
~~~
git clone https://github.com/Matvey021292/laravelTestTask.git
~~~


Install package
------------
~~~
composer install
~~~

Edit database settings .env
------------
~~~
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
~~~


Run Migrations
------------
~~~
php artisan migrate
~~~

Generate a key
------------
~~~
php artisan key:generate
~~~

Start server
------------
~~~
php artisan serve 
~~~
